CSS 一种编译方式

1、对文档做语法分析，然后创建文档树

2、辨认目标的媒体类型(media type)

3、根据媒体类型，从文档树中提取出所有的样式表

4、根据媒体类型给文档树中的所有元素中的所有属性赋予唯一值从而对文档树作注释。属性值根据传递和继承计算。部分数值的计算还需考虑媒体类型给出的样式算法。

5、根据注解好的文档树，生成样式结构。样式结构通常和dom树的结构很像，但需要注意的是当用户使用伪元素和生成的内容时，这两者在一些重要的地方是不同的。首先，样式结构不需要完全像树一样，结构的特点是实现决定的。然后，样式结构可能包括比dom树更多或更少的信息。比如，如果一个在dom树中的元素有一个display:none的属性，这个元素将不会再样式结构中出现。另外，一个列表元素将会生成更多的信息：列表中元素的内容和列表样式信息。CSS用户代理在这个阶段不会改变dom树信息。特别的，

6、将样式结构传递给目标媒体。
